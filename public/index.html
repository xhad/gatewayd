Ripple Gateway Accounts API Reference

Gateway

Users
A user can register or an admin can list/create users. Users can also be maintained in a separate system and referenced with its unique id.

Resource Description
POST /api/v1/gateway/users Create a new user at this gateway.
GET /api/v1/gateway/users/:userId/gateway_account Get the user's account.

Gateway Accounts
Create a new gateway account, which can be associated with a user, ripple addresses, and balances.

Resource Description
POST /api/v1/gateway/users/:userId/gateway_accounts Create a new gateway account given a user id

Balances
Resource to check on user balances in various currencies with the gateway

Resource Description
GET /api/v1/accounts/:accountId/balances Get the balances for a gateway account

Gateway Transactions
List bank transactions made at this gateway

Resource Description
GET /api/v1/gateway/accounts/:accountId/transactions Show gateway transactions from a specific account
GET /api/v1/gateway/transactions/:id Show a single gateway transaction

Gateway Deposits
List bank transactions made at this gateway

Resource Description
POST /api/v1/gateway/accounts/:accountId/deposit Record a deposit into the gateway from outside ripple network

Gateway Withdrawals
List bank transactions made at this gateway

Resource Description
POST /api/v1/gateway/accounts/:accountId/withdrawal Create a withdrawal request out of the gateway to outside the ripple network
GET /api/v1/gateway/withdrawals Show all pending withdrawal requests
POST /api/v1/gateway/withdrawals/:id/accept After disbursing funds outside the ripple network, mark the withdrawal as successful and remove from the pending queue
POST /api/v1/gateway/withdrawals/:id/reject Reject a withdrawal request, destroying the gateway transaction record

Ripple

Ripple Addresses
List Ripple Addresses related to User accounts.

Resource Description
GET /api/v1/accounts/:account_id/ripple_addresses Get the ripple address for a gateway account
POST /api/v1/accounts/:account_id/ripple_addresses/:ripple_address Associate a ripple address with a gateway account

Ripple Transactions
List transactions made on ripple network to the gateway's address.

Resource Description
GET /api/v1/ripple_addresses/:address/ripple_transactions Get all rippple transactions for a ripple address
GET /api/v1/ripple_transactions/:ripple_transaction_id Get any given ripple transaction by id

Ripple Deposits
Send funds into the gateway from outside the ripple network

Resource Description
GET /api/v1/ripple_addresses/:address/deposits List all deposits for an account
POST /api/v1/ripple_addresses/:address/deposits Create a deposit into a gateway account

Ripple Withdrawals
List Bank Transaction Withdrawals

Resource Description
GET /api/v1/ripple_addresses/:address/withdrawals Get a list of a single user's withdrawals to ripple account.
POST /api/v1/ripple_addresses/:address/withdrawals Create a withdrawal from a gateway account to the ripple account.
PUT /api/v1/ripple_transactions/:ripple_transaction_id Update a withdrawal with its status from the ripple payment service.
